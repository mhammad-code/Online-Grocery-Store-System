🛒 Online Grocery Store System
A robust desktop application built with C# WinForms and SQL Server to manage inventory, customers, and sales for a modern grocery retail environment.

📝 Project Description
The Online Grocery Store System is designed to bridge the gap between manual record-keeping and digital inventory management. It provides a user-friendly interface for staff to manage products and for administrators to track sales and stock levels in real-time.

By leveraging ADO.NET for database connectivity, the system ensures data integrity and high performance for CRUD (Create, Read, Update, Delete) operations.

🛠️ Tech Stack
Frontend: C# Windows Forms (WinForms)

Backend: SQL Server (T-SQL)

Data Access: ADO.NET (SqlClient)

IDE: Visual Studio 2022

🚀 Key Features
Inventory Management: Add, update, and delete grocery items with category filtering.

Stock Tracking: Automated stock level updates when an order is placed.

User Authentication: Secure Login and Registration system for different user roles.

Sales Reporting: Track total revenue and order history through the Orders table.

Search Functionality: Real-time product searching using SQL LIKE queries.
